*特性：*

- ☑ symbols：双拼使用大写 S 开头，使用空格 Space 上屏（TODO 是否考虑使用 ~/~ 引入 symbols？是否可以使用 Enter 直接上屏？）；
- ☑ emacs 快捷键：C-n C-p 翻页、C-b C-f 移动光标；
- ☑ custom_phrase.txt 自定义短语（最高的权重）
- ☑ opencc 自定义短语携带 symbols
- ☐ 删词、降权
- ☐ 添加字典，使用简体词库，而不是Rime自带的「明月拼音·简化字」方案（是繁体词库加上一个自动繁转简的设定，由OpenCC转换，用户词典中保存的输入历史还是繁体的）
- ☐ 同步
- ☐ 排序
- ☐ 皮肤
- ☐ 直接schema，不custom


*设计输入法理念:*
- 快速、简洁、易理解、易扩展！
- 不过度配置，不考虑中文输入下的英文输入；
- 输入是一件很快乐的事情，不要让过多的输入规则打断敲击思绪；
