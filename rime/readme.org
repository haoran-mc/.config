*特性：*

- ☑ symbols：双拼使用 ~/~ 开头，使用空格 Space 上屏而不是回车 Enter，回车输入原始输入；
- ☑ emacs 快捷键：C-n C-p 翻页、C-b C-f 移动光标；
- ☑ custom_phrase.txt 自定义短语，通过拼音检索（最高的权重）
- ☑ opencc 自定义短语，通过已有词汇联想（有时候联想失败）
- ☑ 添加字典，使用简体词库，而不是Rime自带的「明月拼音·简化字」方案（是繁体词库加上一个自动繁转简的设定，由OpenCC转换，用户词典中保存的输入历史还是繁体的）
- ☑ 皮肤
- ☑ lua 扩展
- ☑ 直接schema，不custom
- ☑ 同步用户词库：「同步用户词库」→ 词库文件：sync/hashid/dict_*.userdb.txt → 将词库文件复制新机器的 sync/hashid/ 下面 → 「同步用户词库」
- ☒ 中文中的英文输入，或者不需要（不过度配置）
- ☐ 删词、降权（手动操作即可）


*设计输入法理念:*
- 快速、简洁、易理解、易扩展！
- 不过度配置；
- 输入是一件快乐的事情，不要让繁琐的输入规则打断敲击思绪；
