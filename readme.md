**quick start:**

```
git clone --depth 1 git@github.com:haoran-mc/dotfiles.git ~/dotfiles
cd ~/dotfiles
chmod +x ./osx-setup.sh
./osx-setup.sh
```

- code with neovim
- edit with vim


**tools:**

1. terminal: alacritty
2. shell: zsh
3. grep → ripgrep
4. find → fd
5. cat → bat
6. fzf
7. tldr
