# zsh

## 使用 zsh

1. mac 推荐 iterm2, linux 推荐 konsole

    1. 下载 iterm2 [iterm2.com/downloads.html](https://iterm2.com/downloads.html)
    2. iterm2 主题 [iterm2colorschemes](https://iterm2colorschemes.com/)

    <!-- 使用 atom 主题 -->

2. 下载 zsh 并选择 zsh
3. 下载 oh-my-zsh
4. 下载 zsh 插件
5. 下载 powerlevel10k
