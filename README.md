**quick start:**

```
git clone --depth 1 git@github.com:haoran-mc/dotfiles.git ~/dotfiles
cd ~/dotfiles
chmod +x ./install.sh
./install.sh
```

- neovim for code
- vim for edit


**tools:**

1. terminal: iterm2(mac), alacritty(linux)
2. shell: zsh
3. ripgrep instead grep
4. fd instead find
5. bat instead cat
6. fzf
7. tldr
8. bat
